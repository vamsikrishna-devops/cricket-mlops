\# Cricket Win Probability Predictor (MLOps)



An end-to-end beginner-friendly MLOps project that trains a machine learning model to predict \*\*cricket match win probability\*\*, tracks experiments with \*\*MLflow\*\*, serves predictions via \*\*FastAPI\*\*, and runs in \*\*Docker\*\*.



\## Features

\- Model training (scikit-learn Logistic Regression)

\- Experiment tracking with MLflow

\- REST API with FastAPI

\- Dockerized deployment

\- Simple sample dataset for quick start
